## 存活时间

| 环境条件 | 存活时间 |
| --- | --- |
| 印刷品、纸张 | 3小时 |
| 木材、布 | 2天 |
| 玻璃、纸币 | 4天 |
| 不锈钢、塑料 | 7天 |
| 空气 | 2小时-2天 |
| 皮肤 | 8.6小时-22.5小时（不同毒株） |
| 衣服 | 2天<br>聚酯 >3天<br>100%棉 1天<br>涤棉 6小时 |
| 口罩 | 2天-3天 |

### 消毒

| 环境条件 | 存活时间 |
| --- | --- |
| 紫外线 | 60分钟 |
| 75%乙醇 | 2分钟 |
| 0.05%次氯酸钠 | 2分钟 |

### 温度

| 环境条件 | 存活时间 |
| --- | --- |
| 零下 | 几周到几个月 |
| 56°C | 30分钟 |
| 70°C | 5分钟 |
